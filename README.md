## 🚀 Roadmap

Este é um projeto inicial focado na lógica de programação e tratamento de dados em memória. As próximas evoluções planejadas incluem:
- [ ] Implementação de persistência de dados utilizando um Banco de Dados (SQL/NoSQL).
- [ ] Criação de uma interface gráfica (GUI) ou API REST.
- [ ] Sistema de autenticação e criptografia de senhas.
